# PopX

A modern and responsive user authentication application built with React. PopX provides a clean onboarding experience with user registration, login, and profile management interfaces following modern UI/UX principles.

## 🚀 Features

- User Registration
- User Login
- Profile Management
- Responsive Design
- Clean and Modern UI
- Mobile-Friendly Layout
- Reusable React Components
- Fast and Optimized Performance

## 🛠️ Tech Stack

### Frontend
- React.js
- JavaScript (ES6+)
- HTML5
- CSS3

### Tools & Deployment
- Git & GitHub
- Vercel
